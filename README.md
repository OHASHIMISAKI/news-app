# ニュース表示アプリ

## カテゴリ変更から画面表示に至るまでのロジック

### 1.  **ユーザー操作（フロントエンド）**
*   画面上のカテゴリボタン（経済・グルメ・健康）をクリックします。
*   JavaScriptの非同期通信 (fetch API) により、選択されたカテゴリIDをバックエンド (`/api/news`) へ送信します。

### 2.  **データ取得と加工（バックエンド）**
*   Flaskサーバーがリクエストを受け取り、カテゴリに対応したRSS URLを選択します。
*   `feedparser` ライブラリを使用して、対象のRSSから最新ニュースを取得し、上位10件を抽出します。
*   抽出した各ニュースのタイトルを「Gemini API」に送信します。
*   Gemini APIがタイトルを解析し、内容を表す重要なタグを3つ生成して返却します。

### 3.  **画面表示（フロントエンド）**
*   バックエンドは「タイトル」「リンク」「概要」「AI生成タグ」を含んだJSONデータを返却します。
*   JavaScriptがデータを受け取り、動的にニュース一覧（HTML要素）を生成し、画面に表示します。

## 必要な起動手順、必要なインストール作業

### 1. **必要要件**
*   Python 3.9以上 がインストールされていること

### 2. **インストール**

1. プロジェクトフォルダにて、以下のコマンドを実行して必要なライブラリをインストールしてください。

```bash
pip install -r requirements.txt
```
※ requirements.txt がない場合は、以下を個別にインストールしてください。

```bash
pip install flask feedparser google-generativeai
```

2. APIキーの設定
*  app.py（または実行ファイル名）を開き、以下の部分にGemini APIキーを入力してください。

app.py 
GEMINI_API_KEY = "APIキー"

3. アプリケーションの起動
以下のコマンドでサーバーを起動します。

```bash
python app.py
```

4. ターミナルに以下のような表示が出たら起動成功です。
*  Running on http://127.0.0.1:5000

5. ブラウザで http://127.0.0.1:5000 にアクセスしてください。
